# MEMORY.md

## 项目：个人研究助手（research-assistant）

- 目标：围绕“智能物联感知与处理”构建个人研究门户，覆盖文献抓取、实验记录、报告与路由。
- 当前能力：
  - 基础页面与 API 可运行（dashboard/papers/experiments/reports）
  - 模型路由基线已接入
  - arXiv 抓取与自动抓取链路已打通
- 关键改进方向：
  1. 顶刊顶会识别（白名单与规则化）
  2. 抓取质量分层与可视化筛选
  3. 整点/半点进展汇报稳定可见投递

## 协作偏好

- 用户要求：开发期间需按整点与半点固定汇报进展，且必须可见、可验证。
- 汇报格式固定：
  1) 本时段完成
  2) 当前阻塞/风险
  3) 下个半小时计划
  4) 可验证访问路径
