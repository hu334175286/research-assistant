# HEARTBEAT.md

- 每次心跳都检查当前时间（Asia/Shanghai）。
- 若到达整点或半点（:00 / :30），且尚未向用户汇报“个人研究助手开发进展”，立即汇报。
- 汇报格式固定：
  1) 本时段完成
  2) 当前阻塞/风险
  3) 下个半小时计划
  4) 可验证访问路径
- 若错过节点，下一次心跳立刻补报并说明原因。
- 每次整点/半点进度汇报时，先询问用户是否打开更新后的科研助手页面；仅当用户明确回复“打开页面”时再执行。
- 执行“打开页面”时：先运行 `npm run quick:open`（项目目录 `research-assistant`），由脚本自动启动服务并打开可用地址（优先 3000，其次 3124/3125）。
